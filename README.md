# My Homebrew Tap

More caffeine for you

## Install

```sh
# Tap a new formula
brew tap caffeine-addictt/tap
```

```sh
# Install
brew install caffeine-addictt/tap/$BINARY_NAME

# Upgrade
brew upgrade caffeine-addictt/tap/$BINARY_NAME
```

* [waku](https://github.com/caffeine-addictt/waku)
